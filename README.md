## Project Overview
A mini project about trending data jobs to find their sustainability in the future and help candidates stand out by making the right decisions based on key findings.
This project uses Excel for forecasting and analysis.

## Business Problem
Many students get overwhelmed with so many questions and career routes. This project aims to:
- Find the top 5 countries with the highest averages
- Help choose work settings by comparing them to seniority
- Predict the future of the top Data Fields
- The reason for the decline in the Data Analysis field


## Tools and Technologies
- Excel (Time-series, Charts)
- Ms Word (Report)


## Key Findings

1. **Top 2 Data Job Categories:** Machine Learning and AI & Data Science, and Research.
2. **Data Analysis Category:** This field is declining gradually.
3. **Company Preference:** Medium-sized companies are more prone to hiring.
4. **Top 3 countries for Data Analysis field:** Saudi Arabia(30%), United States(19%) & Australia(18%).
5. **Work settings for Entry and Executive Levels:** Remote jobs are more prone to hiring.
6. **Work settings for Mid and Senior Levels:** In-person jobs are more prone to hiring.



## Final Recommendations

- Take more time in career choice by analyzing future job predictions.
- Focus on upskilling.
- Build connections for guidance and knowledge.
- Apply to Medium-sized companies for a higher selection rate.
- Apply to the top 10 job locations around the world.
